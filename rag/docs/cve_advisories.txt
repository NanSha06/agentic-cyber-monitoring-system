CVE SECURITY ADVISORIES — CYBER-PHYSICAL BATTERY SYSTEMS
Compiled from NVD, ICS-CERT, and internal threat intelligence | 2023-2024

CVE-2023-28531 — Remote Code Execution in BMS Firmware
CVSS Score: 9.8 (CRITICAL)
Affected: Battery Management Systems running firmware < v4.2.1
Description: A buffer overflow vulnerability in the Modbus TCP handler allows unauthenticated
remote attackers to execute arbitrary code with system privileges. Active exploitation observed
in the wild targeting industrial BESS installations.
Indicators of Compromise:
- Unusual Modbus TCP traffic on port 502 from external IPs
- Unexpected firmware version changes
- Auth failures followed by successful logins from anomalous IPs
Mitigation: Update firmware to v4.2.1+, restrict port 502 to internal VLANs only,
enable authenticated Modbus (Modbus Security extension RFC 8628).

CVE-2023-41679 — SCADA Authentication Bypass
CVSS Score: 8.1 (HIGH)
Affected: SCADA HMI software versions 2.x - 3.4.x
Description: Improper session management allows attackers to bypass authentication
by replaying captured session tokens. Enables unauthorized write access to control parameters.
Indicators of Compromise:
- auth_failure_burst_rate spike followed by immediate successful session
- Session tokens reused from different source IP
- Unexpected changes to charging/discharging parameters
Mitigation: Upgrade to version 3.5+, implement token rotation, enable MFA.

CVE-2024-21762 — Network Scan Preceding BMS Exploit
CVSS Score: 7.5 (HIGH)
Affected: All BMS with exposed management interfaces
Description: Threat actors perform reconnaissance port scans (portscan threat_type)
as precursor to targeted exploitation of BMS vulnerabilities. Pattern: portscan detected
15-60 minutes before authentication attempts or command injection.
Indicators of Compromise:
- lateral_move_indicator = 1 with preceding portscan events
- Unusual TCP SYN flood to ports 502, 102, 4840 (Modbus, S7, OPC-UA)
- packet_entropy anomaly concurrent with auth_failures
Mitigation: Network segmentation, IDS signatures for BMS protocol ports,
alert on portscan-to-authfail correlation within 60-minute window.

CVE-2023-36884 — DDoS Against BMS Monitoring Layer
CVSS Score: 6.5 (MEDIUM)
Affected: Web-based BMS dashboards with public-facing interfaces
Description: HTTP flood attacks target monitoring dashboards, causing availability loss
and potentially masking simultaneous physical attacks on battery systems.
Indicators of Compromise:
- High packet_entropy combined with ddos threat_type classification
- Dashboard response time >10 seconds
- Normal battery telemetry hidden behind monitoring outage
Mitigation: WAF deployment, rate limiting on monitoring endpoints, offline alerting
channels (SMS/email) that operate independently of web dashboards.
